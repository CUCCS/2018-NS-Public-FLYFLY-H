# 网络扫描

### 准备工作
- 将实验1中的NAT网络换成intnet, 手动为attacker的eth0配置IP,掩码和网关，网关和target网关一样。
```
  allow-hotplug eth0
  iface eth0 inet static
  address 196.168.56.103
  netmask 255.255.255.0
  gateway 196.168.56.101
```

### 实验环境
- attacker：196.168.56.103
- target：196.168.56.102
- gateway: 196.168.56.101
